This is the repository home of the [shpe.ga](https://shpe.ga). It is build on [Jekyll](https://jekyllrb.com/), and the dependencies are managed by [Bundler](http://bundler.io/).

## Setting up

~~~bash
$ bundle install
~~~

## Run it

~~~bash
$ bundle exec jekyll serve
~~~
